Introduction

Welcome to the HTML guide! HTML, or HyperText Markup Language, is the backbone of all websites. This language provides the structure of a webpage, allowing developers to organize text, images, links, and other multimedia elements into a cohesive and navigable format.

  • HTML is the foundation of web pages.
  • HTML structures web content.
  • It supports multimedia elements.
  • Works across all major browsers.
  • Used by all web developers.
What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure webpages. HTML enables developers to describe the purpose and appearance of content on the web, from simple text to complex page layouts. Since its inception, HTML has become the foundational language for web design, supported by all browsers worldwide.

At its core, HTML uses elements—defined by tags—to organize and present content. Each tag provides a unique role, whether to create a heading, a paragraph, an image, or a link. For example, the <h1> tag denotes a major heading, while the <p> tag indicates a paragraph of text.

Core HTML Structure

The structure of an HTML document is composed of several key components. Each HTML document typically begins with a declaration <!DOCTYPE html>, followed by the root element, <html>, which contains all the content on the page.

Within the <html> element, the <head> section holds metadata, such as the document title and links to stylesheets. The <body> section includes all visible content, including text, images, and interactive elements.

By organizing these sections correctly, developers ensure their pages are readable, accessible, and compatible across browsers and devices.

Web Design Basics

Understanding HTML is the first step to designing webpages. HTML provides structure, but it is usually paired with CSS for styling and JavaScript for interactivity. Together, these languages form the core of web development, enabling the creation of visually appealing and functional websites.

HTML Paragraphs

The <p> tag defines a paragraph. Text within this tag will be displayed as a block of text with spacing above and below. Paragraph tags are essential for separating text and making content more readable.

<p>This is a paragraph.</p>
HTML Headings

HTML provides six levels of headings, from <h1> to <h6>. The <h1> tag represents the main heading, while <h6> is the smallest, typically used for sub-subsections. Using headings correctly helps with content organization and accessibility, as search engines and screen readers use these elements to understand the document structure.

<h1>Main Heading</h1>
<h2>Subheading</h2>
HTML Inline Elements

This section explains the difference between block and inline elements and introduces HTML links, which allow for internal and external navigation.

  • Block elements occupy full width.
  • Inline elements fit in-line with text.
  • HTML links navigate pages.
  • Images can be inline or block.
  • Inline styling affects specific elements.
Block vs. Inline Elements

Block elements take up the full width of their container, creating a new line and stacking vertically. Inline elements, on the other hand, take up only as much width as they need, staying within the same line. Understanding this distinction is crucial for arranging content effectively on a webpage.

Examples:

  • Block elements: <div>, <p>, <h1>
  • Inline elements: <span>, <a>, <img>
HTML Links

The <a> tag defines a hyperlink, enabling users to navigate between webpages, documents, or specific locations within the same page. The href attribute specifies the URL or location to link to.

<a href="https://www.example.com">Visit Example</a>
HTML Forms

HTML forms allow users to submit data to a web server, making them essential for user interaction on websites. Forms can include text inputs, radio buttons, checkboxes, and more.

  • The <form> tag creates a form.
  • The <input> tag captures user input.
  • Form elements like text, radio, and checkbox.
  • action attribute specifies the data destination.
  • method attribute defines GET or POST requests.

Example:

<form action="/submit-form" method="POST">
  <label>Name:</label>
  <input type="text" name="name">
  <input type="submit" value="Submit">
</form>
HTML Media

HTML provides tags for embedding multimedia, such as images, audio, and video, allowing for a richer web experience.

  • <img> tag for images.
  • <audio> tag for embedding audio.
  • <video> tag for embedding video.
  • src attribute specifies the media source.
  • controls attribute adds play, pause, and volume options.

Example:

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>
